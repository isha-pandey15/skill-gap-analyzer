# AI Skill Gap Analyzer

AI Skill Gap Analyzer is a simple web application that compares a resume with a job description to identify matching skills, missing skills, and overall job readiness.

This tool helps students and job seekers understand whether they are prepared for a role and what skills they should improve.


# Features

- Upload Resume (PDF or TXT)
- Paste Job Description
- Extract Skills from Resume
- Identify Matching Skills
- Detect Missing Skills
- Calculate Skill Match Score
- Suggest Skill Improvements
- Download Analysis Report (PDF)

# Why This Project is Useful

Many applicants apply for jobs without knowing if their skills match the requirements. This tool helps users:

- evaluate job readiness  
- identify skill gaps  
- improve resumes  
- plan learning goals  

## Tech Stack

- Python
- Streamlit
- PyPDF2
- Regex (Skill Extraction)
- ReportLab (PDF Generation)
- Hugging Face Inference API *(optional AI enhancement)*
